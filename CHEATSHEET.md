# Cheatsheet

## neovim
Terminal text editor. Steep learning curve but extremely fast once learned. Using kickstart.nvim as the starting config.

| Command | Description |
|---------|-------------|
| `v` | Open Neovim |
| `v <file>` | Open a specific file |
| `nv` | Open Neovide (GUI version) |
| `:checkhealth` | Verify everything is set up correctly |

---

## yazi
Terminal file manager — navigate directories visually, preview images/code inline (uses Kitty Graphics Protocol), bulk rename and manage files without leaving the terminal.

| Command | Description |
|---------|-------------|
| `y` | Open yazi in current directory |
| `yy` | Open yazi and cd into the last directory when you quit |
| `o` | Create a new file (add `/` at end for directory) |
| `dd` | Trash selected file(s) |
| `yy` (inside yazi) | Copy selected file(s) |
| `Space` | Select/deselect file |
| `Enter` | Open file |
| `Esc` | Quit |

---

## lazygit
Terminal UI for git — stage files, browse history, resolve conflicts, rebase and squash commits without memorizing git commands.

| Command | Description |
|---------|-------------|
| `lg` | Open lazygit in the current repo |
